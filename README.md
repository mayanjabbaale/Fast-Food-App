# Fast Food App

A modern mobile fast food ordering application built with Expo and TypeScript. Users can browse menus, add items to cart, manage orders, and complete purchases through an intuitive mobile interface.

## Key Features

- User authentication and profile management
- Browse and search through food menus
- Add items to cart with quantity management
- Secure checkout process
- Order history and tracking
- Responsive design optimized for mobile devices

## Tech Stack

- **Framework**: Expo (React Native)
- **Language**: TypeScript
- **Styling**: NativeWind (Tailwind CSS for React Native)
- **Backend**: Appwrite
- **State Management**: Custom stores (Zustand-style)
- **Navigation**: Expo Router
